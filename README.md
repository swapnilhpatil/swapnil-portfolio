## 🛠️ The Playbook (Running locally)

Want to see how the sausage is made? Clone this repo and spin it up on your own machine. 

1.  **Clone the repo:**
    ```bash
    git clone https://github.com/swapnilhpatil/swapnil-portfolio.git
    cd swapnil-portfolio
    ```

2.  **Install the dependencies:**
    *(Grab a coffee, this might take a minute).*
    ```bash
    npm install
    ```

3.  **Start the dev server:**
    ```bash
    npm run dev
    ```

4.  **Open your browser:**
    Navigate to [http://localhost:3000](http://localhost:3000) and watch the magic happen.
